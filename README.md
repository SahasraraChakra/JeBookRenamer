**Usage Instructions**

1. Command Buttons are Self Expalining
2. Add files to list.
3. To preview the file in _System viewer_, select the file from the list and click on **Open**
4. Double click it to set to **Edit*
5. As you change Topic if it is in the list, **next possible Topic** will be **highlighted automatically** in the dropdown.
6. Use **Arrow keys to navigate** to Topic of choice
7. If you Topic is new, moving out of combobox will automatically **add the New topic to list**
8. And as you type, the new name will be automatically prepared and will be visible in **Preview Name**
9. Enable checkbox **Auto Edit Next** to set next item in the list automatically to edit mode.
10. Click on **Set Name**, to set the new prepared name to the files list
11. Use **Rename Selected** _or_ **Rename All** to rename files in the list.
12. If you want to **create new folder** for each topic enable checkbox **Create Topic Folder(s)**
13. If you want to **move the renamed files** to another folder than original folder, enable checkbox **Move to Folder** and _**Select Folder**_ using **Browse** button


Note :

_Moving files may replace files with same name in the target folder_

_Renaming may fail if file name already exists._
